# 需求文档

## 1. 应用概述

### 1.1 应用名称
空间纪实叙事互动游戏

### 1.2 应用描述
一款基于微信小程序的空间叙事互动应用，1:1复刻南京大学鼓楼校区实景地图，玩家可在地图上标记点位、留言、上传照片记录生活故事，并通过点位关联串联形成完整叙事线。其他玩家可查看单个点位故事或沉浸式翻阅连贯的人生故事篇章。

## 2. 用户与使用场景

### 2.1 目标用户
- 南京大学鼓楼校区师生、校友
- 对校园生活有记录与分享需求的用户
- 希望了解他人校园故事的用户

### 2.2 核心使用场景
- 用户在校园特定地点标记并记录个人经历与故事
- 用户通过关联多个点位梳理行轨迹与事件脉络
- 用户浏览他人留下的单点故事或完整故事线

## 3. 页面结构与功能说明

### 3.1 页面结构

```
├── 地图主页
│   ├── 地图展示区域
│   ├── 点位标记功能
│   └── 点位查看功能
├── 点位详情页
│   ├── 单点位信息展示
│   └── 关联点位故事线展示
└── 点位创建/编辑页
    ├── 留言输入
    ├── 照片上传
    └── 点位关联设置
```

### 3.2 地图主页

#### 3.2.1 地图展示
- 使用微信小程序官方配置地图组件
- 1:1复刻南京大学鼓楼校区实景地图
- 支持地图缩放与拖动操作

#### 3.2.2 点位标记显示规则
- 地图放大至50%时，标记点以脚印图标显示
- 地图放大至100%时，标记点以创建者微信头像显示

#### 3.2.3 点位标记功能
- 用户长按地图任意位置，触发标记点创建
- 跳转至点位创建页面

#### 3.2.4 点位查看功能
- 用户点击地图上任意标记点，跳转至点位详情页

### 3.3 点位创建/编辑页

#### 3.3.1 留言输入
- 提供文本输入框，用户撰写该点位的故事与经历

#### 3.3.2 照片上传
- 支持用户上传照片，记录该点位的视觉内容

#### 3.3.3 点位关联设置
- 用户可选择已创建的其他点位进行关联
- 关联后形成点位连线，构建完整故事线
- 支持设置关联顺序，梳理行轨迹与事件脉络

#### 3.3.4 保存与提交
- 用户完成内容填写后，保存点位信息
- 点位信息包含：地理位置、留言内容、上传照片、创建者微信头像、关联点位列表

### 3.4 点位详情页

#### 3.4.1 单点位信息展示
- 显示该点位的留言内容
- 显示该点位上传的照片
- 显示创建者微信头像
- 显示点位地理位置信息

#### 3.4.2 关联点位故事线展示
- 若该点位存在关联点位，显示完整故事线入口
- 用户点击故事线入口，按关联顺序依次展示各点位内容
- 支持在故事线中跳转至任意关联点位
- 地图上显示点位连线，标识故事线路径

## 4. 业务规则与逻辑

### 4.1 点位标记规则
- 每个用户可在地图上创建多个标记点
- 每个标记点必须包含地理位置信息
- 标记点创建后自动关联创建者微信头像

### 4.2 点位关联规则
- 用户仅可关联自己创建的点位
- 关联点位数量不限
- 关联顺序由用户自定义设置
- 关联关系形成后，在地图上以连线方式呈现

### 4.3 点位查看规则
- 所有用户均可查看地图上任意标记点
- 查看单点位时，展示该点位完整信息
- 查看故事线时，按关联顺序依次展示各点位内容

### 4.4 地图缩放显示规则
- 地图缩放比例小于50%时，标记点不显示
- 地图缩放比例达到50%时，标记点以脚印图标显示
- 地图缩放比例达到100%时，标记点以创建者微信头像显示

## 5. 异常与边界情况

| 场景 | 处理方式 |
|------|----------|
| 用户未授权微信头像 | 使用默认头像占位 |
| 用户未填写留言内容 | 提示用户留言为必填项，禁止保存 |
| 用户未上传照片 | 允许保存，照片为可选项 |
| 用户关联点位时选择不存在的点位 | 系统过滤无效关联，仅保存有效点位 |
| 地图加载失败 | 提示用户网络异常，引导重新加载 |
| 点位信息保存失败 | 提示用户保存失败，引导重试 |

## 6. 验收标准

1. 用户打开小程序，进入地图主页，地图正确显示南京大学鼓楼校区实景
2. 用户长按地图任意位置，成功创建标记点并跳转至点位创建页面
3. 用户在点位创建页面填写留言、上传照片，保存后返回地图主页
4. 用户在地图主页放大至50%，标记点以脚印图标显示
5. 用户在地图主页放大至100%，标记点以创建者微信头像显示
6. 用户点击标记点，进入点位详情页，查看留言、照片及创建者头像
7. 用户创建第二个标记点，在点位创建页面关联第一个点位，保存后地图显示点位连线
8. 其他用户点击关联点位，进入点位详情页，点击故事线入口，按顺序查看完整故事线内容
